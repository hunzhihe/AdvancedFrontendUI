# 语言切换功能 — 完整数据流程

## 概览图 (Mermaid)

```mermaid
mindmap
  root((语言切换功能))
    初始化阶段
      Widget_OptionsScreen::NativeOnInitialized
        绑定 OnRegistryNewOptionsData 委托 → RefreshLocalizedDataRegistry
        绑定选项卡选择、列表悬停/选中事件
      NativeOnActivated
        GetOrCreateDataRegistry
          InitOptionsDataRegistry
            InitGameplayCollectionTab
              GameDifficulty → AddDynamicOption + SetDescriptionLocalizationKey + SetDisplayNameLocalizationKey
              Language → AddDynamicOption + SetDisplayNameLocalizationKey
              所有数据对象设置本地化键
            InitAudioCollectionTab
              OverallVolum, MusicVolume... → SetDisplayNameLocalizationKey
              AllowBackgroundAudio → SetTrue/FalseLocalizationKey
            InitVideoCollectionYab
              WindowMode → AddEnumOptions（枚举名作键）
              质量设置 → SetQualityLocalizationKeys 辅助函数
              所有数据对象设置本地化键
            InitControlCollectionTab
              按键映射 → 不需要本地化
            RefreshAllLocalizedText（末尾统一调用）
              遍历所有数据对象 → RefreshLocalizedText
                基类：查 StringTable 填充 Description + DisplayName
                String 子类：填充选项显示文本数组
                Stringbool：先应用 True/FalseLocalizationKey
                StringResolution：仅基类，跳过选项文本
        注册选项卡 → RequestRegisterTab
        SetListItems → 创建条目控件 → OnOwningListDataObjectSet
    运行时切换
      触发方式
        上一项/下一项按钮
          BackToPerviousOption / AdvanceToNextOption
        手柄 Rotator
          OnRotatorValueChanged → OnRotatorInitiatedValueChange
      值变更
        DataDynamicSetter->SetValueFromString
        UFrontendUIGameUserSettings::SetCurrentLanguage
      OnRegistryNewOptionsData.ExecuteIfBound
        RefreshLocalizedDataRegistry
    刷新阶段
      RefreshAllLocalizedText
        TabCollection 自身 → RefreshLocalizedText
        递归子级 → RefreshChildListDataRecursively
        每个数据对象 → RefreshLocalizedText
          按类型分发
            ListDataObject_Base
              刷新 DescriptionRichText ← DescriptionLocalizationKey
              刷新 DataDisplayName ← DisplayNameLocalizationKey
            ListDataObject_String
              先调基类刷新描述和显示名称
              FrameRateLimit → 跳过（直接 return）
              其他 → 清空显示文本数组，重新查 StringTable 填充
              键优先级：LocalizationKeyArray[i] > OptionString
              TrySetDisplayTextByStringValue 刷新当前选中
            ListDataObject_Stringbool
              先应用 TrueLocalizationKey / FalseLocalizationKey
              再调父类（父类用键查表填充）
            ListDataObject_StringResolution
              仅调 UListDataObject_Base::RefreshLocalizedText
              跳过选项值文本刷新
            ListDataObject_StringEnum
              继承父类逻辑，枚举名作键
            ListDataObject_StringInteger
              继承父类逻辑，SetLocalizationKeyForIntegerValue 覆盖键
      刷新选项卡按钮
        遍历 RegisteredOptionsTabCollections
        获取按钮 → SetButtonText(GetDataDisplayName)
      刷新列表条目控件
        遍历 ListView 所有条目
        获取 EntryWidget → 调用 OnOwningListDataObjectSet
        子类重新读取数据对象文本
          Widget_ListEntry_String → PopulateTextLabels + SetSelectedOptionByText
          Widget_ListEntry_Base → SetText(DataDisplayName)
      刷新详情视图
        UpdateDetailView 当前选中项
    数据对象层
      三个并行数组
        AvaiableOptionsStringArray（值）
          例：["true", "false"]、["Easy", "Medium", "Hard"]、["0", "1", "2"]
        AvaiableOptionsDisplayTextArray（显示文本）
          语言切换时清空重填
        AvaiableOptionsLocalizationKeyArray（本地化键）
          空 = 用 OptionString 作键
          非空 = 用自定义键
      本地化键来源
        DisplayNameLocalizationKey → 基类 RefreshLocalizedText
        DescriptionLocalizationKey → 基类 RefreshLocalizedText
        OptionString（默认）→ String 子类 RefreshLocalizedText
        SetOptionLocalizationKey → 自定义每项键
        SetTrue/FalseLocalizationKey → Stringbool
        SetLocalizationKeyForIntegerValue → StringInteger
      StringTable 查表
        UFrontendUIFunctionLibrary::GetCurrentLanguageTextFromTable
        当前语言 → 对应 StringTable 资产
        English → ST_OptionsScreenDescription
        ZH_Ch → ST_ZH_OptionScreenDescription
      StringTable 查表
        UFrontendUIFunctionLibrary::GetCurrentLanguageTextFromTable
        当前语言 → 对应 StringTable 资产
        English → ST_OptionsScreenDescription
        ZH_Ch → ST_ZH_OptionScreenDescription
    不需要本地化的选项
      FrameRateLimit
        AddDynamicOption 中跳过查表
        RefreshLocalizedText 中直接 return
      ScreenResolution
        AddDynamicOption 中跳过查表
        RefreshLocalizedText 重写仅调基类
    键管理策略
      直接用 OptionString 作键
        GameDifficulty → "Easy", "Medium", "Hard"
        Language → "English", "ZH_Ch"
        WindowMode → "Fullscreen", "WindowedFullscreen", "Windowed"
      自定义独立键
        Stringbool → DN_Enable, DN_Disabled, DN_ON, DN_OFF
        质量等级 → DN_Quality_Low ~ DN_Quality_Cinematic
        显示名称 → DN_GameDifficulty, DN_Language...
        描述文本 → GameDifficulty, WindowModeDescKey...
```

## 详细数据流程

### 一、初始化阶段

```
应用启动
│
├─ UWidget_OptionsScreen::NativeOnInitialized()
│   ├─ 注册 UI 操作绑定（重置、返回）
│   ├─ 绑定选项卡选择事件
│   ├─ 绑定列表悬停/选中事件
│   └─ 绑定 OnRegistryNewOptionsData 委托 → RefreshLocalizedDataRegistry ★
│
├─ UWidget_OptionsScreen::NativeOnActivated()
│   │
│   └─ GetOrCreateDataRegistry() → InitOptionsDataRegistry()
│       │
│       ├─ InitGameplayCollectionTab()
│       │   ├─ GameplayTabCollection    → SetDisplayNameLocalizationKey("DN_Gameplay")
│       │   ├─ GameDifficultyOption     → SetDisplayNameLocalizationKey("DN_GameDifficulty")
│       │   │                             SetDescriptionLocalizationKey("GameDifficulty")
│       │   │                             AddDynamicOption("Easy", ...), AddDynamicOption("Medium", ...)
│       │   ├─ TestItemOption           → SetDisplayNameLocalizationKey("DN_TestImageItem")
│       │   └─ Language                 → SetDisplayNameLocalizationKey("DN_Language")

│       │
│       ├─ InitAudioCollectionTab()

│       │   └─ SoundCategoryCollection   → DN_Sound
│       │       ├─ AllowBackgroundAudio  → DN_AllowBackgroundAudio
│       │       │   SetTrueLocalizationKey("DN_Enable")

│       │           SetTrueLocalizationKey("DN_Enable")

### 二、运行时语言切换

```
用户在 Language 选项上操作:
│
├─ [键盘/鼠标] 点击 ← → 按钮
│   ├─ OnPreviousOptionButtonClicked() / OnNextOptionButtonClicked()
│   │   └─ CachedOwningStringDataObject->BackToPerviousOption()
│   │       │
│   │       ├─ CurrentStringValue = 新值
│   │       ├─ DataDynamicSetter->SetValueFromString(新值)
│   │       │   └─ UFrontendUIGameUserSettings::SetCurrentLanguage(新语言)
│   │       └─ NotifyListDataModified()  ← 通知修改
│   │
│   └─ [如果是 Language 选项]
│       └─ OnRegistryNewOptionsData.ExecuteIfBound() ★
│
├─ [手柄] Rotator 旋转
│   └─ OnRotatorValueChanged()
│       └─ OnRotatorInitiatedValueChange()
│           ├─ DataDynamicSetter->SetValueFromString(新值)
│           │   └─ SetCurrentLanguage(新语言)
│           └─ [如果是 Language 选项]
│               └─ OnRegistryNewOptionsData.ExecuteIfBound() ★
│
└─ ★ RefreshLocalizedDataRegistry()  ← 核心刷新入口
    │
    ├─ 1. CreatedOwningDataRegistry->RefreshAllLocalizedText()
    │   │
    │   遍历所有选项卡集合及子数据对象:
    │   └─ 每个数据对象 → RefreshLocalizedText()
    │       │
    │       ├─ [UListDataObject_Base 层]
    │       │   ├─ DescriptionLocalizationKey 非空?
    │       │   │   → GetCurrentLanguageTextFromTable(当前语言, Key)
    │       │   │   → SetDescriptionRichText(本地化文本)
    │       │   └─ DisplayNameLocalizationKey 非空?
    │       │       → GetCurrentLanguageTextFromTable(当前语言, Key)
    │       │       → SetDataDisplayName(本地化文本)
    │       │
    │       ├─ [UListDataObject_String 层]
    │       │   ├─ 调 Super → 基类刷新描述+显示名称
    │       │   ├─ FrameRateLimit? → return（跳过）
    │       │   ├─ 清空 AvaiableOptionsDisplayTextArray
    │       │   ├─ 确保 LocalizationKeyArray 大小一致
    │       │   └─ 遍历 AvaiableOptionsStringArray:
    │       │       ├─ LookupKey = LocalizationKeyArray[i]（非空） ?: OptionString
    │       │       ├─ GetCurrentLanguageTextFromTable(当前语言, LookupKey)
    │       │       └─ 填充到 AvaiableOptionsDisplayTextArray
    │       │   └─ TrySetDisplayTextByStringValue(CurrentStringValue)
    │       │
    │       ├─ [UListDataObject_Stringbool 层]
    │       │   ├─ 应用 TrueLocalizationKey → SetOptionLocalizationKey("true", Key)
    │       │   ├─ 应用 FalseLocalizationKey → SetOptionLocalizationKey("false", Key)
    │       │   └─ Super → 走 String 层逻辑（用刚设置的键查表）
    │       │
    │       ├─ [UListDataObject_StringResolution 层]
    │       │   └─ 仅调 UListDataObject_Base::RefreshLocalizedText()
    │       │      跳过选项文本刷新（分辨率字符串不需要本地化）
    │       │
    │       ├─ [UListDataObject_StringEnum 层]
    │       │   └─ 继承 String 层逻辑，枚举名(Fullscreen...) 作为备选键
    │       │
    │       └─ [UListDataObject_StringInteger 层]
    │           └─ 继承 String 层逻辑，SetLocalizationKeyForIntegerValue 覆盖的键优先
    │
    ├─ 2. 刷新选项卡按钮文本
    │   遍历 RegisteredOptionsTabCollections:
    │   └─ GetTabButtonBaseByID → SetButtonText(TabCollection->GetDataDisplayName())
    │
    ├─ 3. 刷新列表条目控件
    │   遍历 CommonListView_OptionsList->GetListItems():
    │   └─ GetEntryWidgetFromItem → OnOwningListDataObjectSet(DataObject)
    │       ├─ [UWidget_ListEntry_Base]
    │       │   └─ SetText(DataDisplayName)  ← 更新标题
    │       └─ [UWidget_ListEntry_String]
    │           ├─ PopulateTextLabels(AvaiableOptionsDisplayTextArray)  ← 重建选项列表
    │           └─ SetSelectedOptionByText(CurrentDisplayText)         ← 更新选中项
    │
    └─ 4. 刷新详情视图
        └─ DetailView_ListEntryInfo->UpdateDetailView(SelectedItem)
            └─ 重新读取 DescriptionRichText 等属性
```

### 三、数据对象中的三个并行数组

```
以 AllowBackgroundAudio 为例（Stringbool，语言=中文时）:

AvaiableOptionsStringArray:         ["true",            "false"]
AvaiableOptionsLocalizationKeyArray: ["DN_Enable",       "DN_Disabled"]
AvaiableOptionsDisplayTextArray:     ["启用",             "禁用"]

以 OverallQuality 为例（StringInteger，语言=中文时）:

AvaiableOptionsStringArray:        ["0",              "1",              "2",            "3",            "4"]
AvaiableOptionsLocalizationKeyArray: ["DN_Quality_Low","DN_Quality_Normal","DN_Quality_High","DN_Quality_Epic","DN_Quality_Cinematic"]
AvaiableOptionsDisplayTextArray:    ["低",              "中",              "高",            "史诗",           "电影"]

以 GameDifficulty 为例（String，语言=中文时，直接用 OptionString 作键）:

AvaiableOptionsStringArray:         ["Easy",    "Medium",   "Hard"]
AvaiableOptionsLocalizationKeyArray: ["",        "",         ""]      ← 空 = 用 OptionString 作键
AvaiableOptionsDisplayTextArray:     ["简单",     "普通",     "困难"]
```

### 四、StringTable 查表流程

```
UFrontendUIFunctionLibrary::GetCurrentLanguageTextFromTable(Language, Key)
│
├─ Language == English
│   └─ 查表: /Game/.../ST_OptionsScreenDescription.ST_OptionsScreenDescription
│       └─ 返回 Key 对应的英文文本
│
└─ Language == ZH_Ch
    └─ 查表: /Game/.../ST_ZH_OptionScreenDescription.ST_ZH_OptionScreenDescription
        └─ 返回 Key 对应的中文文本
```

### 五、不需要本地化的特殊选项

| 选项 | 类型 | 跳过方式 |
|------|------|---------|
| FrameRateLimit | UListDataObject_String | AddDynamicOption 中 DataID 检查 → 直接用原始文本; RefreshLocalizedText 中 DataID 检查 → return |
| ScreenResolution | UListDataObject_StringResolution | AddDynamicOption 中 DataID 检查 → 直接用原始文本; RefreshLocalizedText 重写 → 仅调基类, 不刷新选项数组 |

### 六、本地化键管理策略一览

| 用途 | 键示例 | 设置方式 | 适用类型 |
|------|--------|---------|---------|
| 显示名称 | DN_GameDifficulty | SetDisplayNameLocalizationKey | 所有数据对象 |
| 描述文本 | GameDifficulty | SetDescriptionLocalizationKey | 有描述的数据对象 |
| 选项值(直接) | Easy, Medium, Hard | OptionString 即键 | GameDifficulty, Language |
| 选项值(枚举) | Fullscreen, Windowed... | 枚举名即键 | WindowMode (StringEnum) |
| 选项值(Boolean) | DN_Enable, DN_Disabled | SetTrue/FalseLocalizationKey | Stringbool 子类 |
| 选项值(整数) | DN_Quality_Low... | SetLocalizationKeyForIntegerValue | StringInteger 子类 |
| 系统值(不本地化) | — | AddDynamicOption 跳过 | FrameRateLimit, ScreenResolution |
```
